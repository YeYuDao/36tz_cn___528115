<br/></br>### 课程介绍:
验证码一直是互联网网站或平台作为“图灵测试”的重要技术手段，主要用于防护网站不被恶意脚本（爬虫及爆破脚本）读取或测试网站数据。互联网中最为常见的图形验证码有如下4种:短信验证码、图形验证码、图像验证码、行为验证码。
